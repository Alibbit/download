# AliceGPT

跨平台 AI 助手，支持 macOS、Windows、Linux、Android、iOS 全平台。

## ✨ 特性

- 🌍 **全平台支持**：macOS、Windows、Linux、Android、iOS
- 🤖 **多模型支持**：Claude、ChatGPT、Gemini、Ollama、OpenAI 兼容
- 🔌 **MCP 支持**：Model Context Protocol 集成
- 🎨 **Artifact 支持**：代码生成与预览
- 💬 **统一界面**：多模型统一对话体验
- 🔒 **本地**：数据本地存储，保护隐私

## 🚀 快速开始

### 桌面端

下载对应平台的安装包：

- [macOS](https://github.com/Alibbit/download/releases)
- [Windows](https://github.com/Alibbit/download/releases)
- [Linux](https://github.com/Alibbit/download/releases)

### 移动端

- **Android**：从 [Releases](https://github.com/Alibbit/download/releases) 下载 APK
- **iOS**：[App Store](https://apps.apple.com/us/app/alibbit/id6754900796)

## 📖 使用指南

1. 下载并安装对应平台的客户端
2. 在设置中配置 API Key（OpenAI、Anthropic、Google 等）
3. 开始对话
